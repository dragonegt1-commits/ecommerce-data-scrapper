# Ecommerce Data Scraper

This project extracts product data (name, price) from an ecommerce website using Python.

## Features
- Pagination (multiple pages)
- Data cleaning
- Price filtering (< 50,000)
- Top 5 cheapest products identified
- Export to CSV using pandas

## Tech Used
- Python
- BeautifulSoup
- Requests
- Pandas

## Output
Structured CSV file ready for analysis
